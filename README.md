# Developer Portfolio

A modern, responsive developer portfolio website built with React.js.

## Features

- **About Section**: Introduction and profile picture
- **Skills Section**: Showcase of technical skills
- **Projects Section**: Display of key projects
- **Contact Section**: Contact information and links
- **Responsive Design**: Works on all devices

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm

### Installation

1. Clone the repository or download the files.
2. Navigate to the project directory:
   ```
   cd Portfolio
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

- Update personal information in the component files (e.g., name, bio, skills, projects).
- Replace placeholder images with your own.
- Modify styles in the CSS files for branding.

## Build for Production

```
npm run build
```

This will create a `build` folder with optimized files.

## Technologies Used

- React.js
- CSS3
- HTML5
